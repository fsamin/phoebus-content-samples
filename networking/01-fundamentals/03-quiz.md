# Network Fundamentals Quiz

## [multiple-choice] What is a computer network?

- [ ] A single computer connected to the internet
- [x] Two or more devices connected together to exchange data
- [ ] A software application for browsing the web
- [ ] A wireless router

> **Explanation:** At its simplest, a network is just two or more devices connected so they can communicate and share data.

## [multiple-choice] What type of network covers a building or campus?

- [ ] PAN (Personal Area Network)
- [x] LAN (Local Area Network)
- [ ] WAN (Wide Area Network)
- [ ] Internet

> **Explanation:** A LAN (Local Area Network) covers a limited area like an office building or campus. Home Wi-Fi networks and office networks are both LANs.

## [multiple-choice] What is a MAC address?

- [ ] The IP address of a Mac computer
- [x] A unique hardware address assigned to every network interface at the factory
- [ ] A software-configured address that changes when you move networks
- [ ] The password of a Wi-Fi network

> **Explanation:** A MAC (Media Access Control) address is a 48-bit address burned into every network interface card. It is unique worldwide and used for communication within a local network.

## [multiple-choice] Why is data broken into packets?

- [ ] Because the network cable is too thin for large data
- [ ] To encrypt the data
- [x] So multiple conversations can share the same network without blocking each other
- [ ] Because servers have limited memory

> **Explanation:** Packets allow the network to interleave data from many senders. Without packets, one large transfer would block the entire network until it completes.

## [multiple-choice] What does a router do?

- [ ] Connects devices within a local network
- [x] Connects different networks together and decides the best path for data
- [ ] Provides Wi-Fi signal
- [ ] Blocks viruses

> **Explanation:** A router connects different networks (e.g., your LAN to the internet) and makes routing decisions — choosing the best path for each packet to reach its destination.

## [short-answer] In the client-server model, what is the device that requests a service called?

    client

> **Explanation:** The client requests a service (e.g., a browser requesting a web page), and the server provides it (e.g., a web server sending the page back).

## [multiple-choice] How many layers does the TCP/IP model have?

- [ ] 3
- [x] 4
- [ ] 5
- [ ] 7

> **Explanation:** The TCP/IP model has 4 layers: Network Access, Internet, Transport, and Application. Each layer handles one aspect of communication and builds on the layer below it.
