# The Bastion

In production environments, you never connect directly to servers. You go through a **bastion** — a hardened gateway that authenticates, authorizes, and logs every connection. This module introduces The Bastion by OVH, an open-source solution used in PCI-DSS and ISO 27001 certified environments.
